# SRT バイリンガル翻訳コマンド

英語の SRT ファイルを読み込み、日本語翻訳を追加したバイリンガル SRT を出力します。

## 入力ファイル
$ARGUMENTS

## 指示

1. 指定された SRT ファイルを読み込む
2. 各セグメントの英語テキストを自然な日本語に翻訳
3. 以下の形式でバイリンガル SRT を生成:

```
1
00:00:01,000 --> 00:00:03,500
Hello, how are you?
こんにちは、お元気ですか？

2
00:00:04,000 --> 00:00:06,000
I'm fine, thank you.
元気です、ありがとう。
```

4. 元のファイル名に `_bilingual` を付けて保存（例: `video_bilingual.srt`）

## 翻訳ルール

- 字幕なので簡潔に
- 自然な日本語で（直訳NG）
- 口語的な表現を維持
- タイムスタンプは絶対に変更しない
- 英語が上、日本語が下の順序を守る
